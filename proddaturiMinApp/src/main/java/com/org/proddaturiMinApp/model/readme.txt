User Entity :
    id           = To save the User id
    mobileNumber = To save the User mobileNumber
    userName     = To save the UserName

Product Entity :
    productId          = To save the product id.
    productName        = To save the User productName.
    productType        = To save the productType.
    productDescription = To save the product details.
    productImage       = To store the product image.
    price              = To store the product price.
    stock              = To store the product available stock.
